# Account Page

## Layout

Vertical

- shared header
- "Account" (h1, align left)
- Profile Section
- "Sign Out" button

## Profile Section

- "Profile" (h3, align left), Edit Icon (Align right, enter Edit Mode)
- First Name input
- Last Name input
- Email input
- Phone input
- "Cancel" and "Save" buttons (visible in edit mode)

### Interaction

- validate email and phone (Canadian phone number) upon saving
- exit edit mode after cancel or save
